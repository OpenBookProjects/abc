review

就很神奇, 有机会一定要看看原文图书是什么样的; 很难想象的到, 一本欧洲,希腊出的 Python 入门图书, 竟然和我们随处可补习班里老师出的教材那么的气质相通:

全彩印刷，讲解细致，寓教于乐，附赠教学辅导视频250 多道判断题、100 道选择题和 100 道复习题，边学边练，学以致用

这是封面的说明, 一下子就将自己从技术图书摘了出去, 进入畅销爆款教辅的行列中了;

上手一看果然:

  • 全彩印刷,主要设计都在每一页的页头, 引用框花边儿, 以及各种卡通形象上了
  • 讲解细致, 一个语法概念可以用10页来反复练习
  • 教学视频, 镜头语言严格复刻了疫情期间民间自发形成的学校直播风
  • ...

而且, 是首次在一部 Python 入门图书中见到 OOP 内容章节, 这也很神奇, 回想以往看的所有图书, 好象真的, 除了专门论述设计模型以及 Python OOP 开发的专著, 其它入门从来没人专门章讲解的...

全书字体丰富, 使用超过6种字体来区分不同性质的内容; 排版舒朗不伤视力; 练习题丰富, 针对两国有关考试进行了设计;

可以拿来作一个线下兴趣班的教材.

本作品采用知识共享署名-相同方式共享 4.0 国际许可协议进行许可;-)
